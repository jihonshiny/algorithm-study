# 2진수 → 10진수 변환

## 코드 흐름

| input | bit(%10) | di | sum |
|-------|---------|-----|-----|
| 101110 | 0 | 1 | 0 |
| 10111 | 1 | 2 | 2 |
| 1011 | 1 | 4 | 6 |
| 101 | 1 | 8 | 14 |
| 10 | 0 | 16 | 14 |
| 1 | 1 | 32 | 46 |
| 0 | break! | - | 46 |

## 배운 점
- % 10으로 끝자리 뽑고 / 10으로 제거한다
- di는 2⁰부터 시작해서 2배씩 증가한다
- 뒤에서부터 자리를 뽑기 때문에 di가 작은 것부터 시작한다
