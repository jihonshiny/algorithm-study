# 2진수 → 10진수 변환

## 말로 풀어서 이해하기

우리가 종이로 2진수 계산할 때
101110 = 1×32 + 0×16 + 1×8 + 1×4 + 1×2 + 0×1 = 46

코드는 이걸 뒤에서부터 하는 것!

1. 맨 뒤 숫자가 1이야? 0이야? → % 10 으로 확인
2. 1이면 자리값을 sum에 더함, 0이면 안더함 → * di 로 해결  
3. 자리값은 2배씩 커짐 → di * 2
4. 확인한 자리는 지워버림 → / 10
5. 다 지웠으면 끝 → input == 0 이면 break

한 줄 요약
뒤에서부터 한 자리씩 떼서, 1이면 자리값 더하고,
0이면 그냥 넘기고, 자리값은 2배씩 키워가는 것!

## 계산 흐름표

| input | bit(%10) | di | sum |
|-------|---------|-----|-----|
| 101110 | 0 | 1 | 0 |
| 10111 | 1 | 2 | 2 |
| 1011 | 1 | 4 | 6 |
| 101 | 1 | 8 | 14 |
| 10 | 0 | 16 | 14 |
| 1 | 1 | 32 | 46 |
| 0 | break! | - | 46 |
